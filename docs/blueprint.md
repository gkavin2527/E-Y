# **App Name**: Modish

## Core Features:

- Product Browsing (Men & Women): Browse clothing items categorized by men's and women's sections.
- Category Navigation: Navigate through categories such as topwear, bottomwear, footwear, etc.
- Product Detail View: View detailed product information (name, price, category, size options, images, description, rating, stock, ID).
- Search Functionality: Search for specific products or categories.
- Shopping Cart: Add and manage selected items in a shopping cart.
- User Profile: Manage user account information.
- AI Style Tool: Generative AI tool providing style recommendations based on browsing history.

## Style Guidelines:

- Primary color: Soft grey (#808080) for a modern, neutral feel.
- Background color: White (#FAFAFA) for a clean aesthetic.
- Accent color: Deep teal (#008080) for highlights and calls to action.
- Body and headline font: 'Inter' (sans-serif) for a modern, machined, objective, neutral look.
- Simple, minimalist icons for navigation and user actions.
- Clean, spacious layout similar to Zara/H&M with high-quality product images.
- Subtle transitions and animations for a smooth user experience.
